# External Sort Demo
Chương trình minh họa thuật toán External Sort cho file nhị phân chứa số double.

## Chức năng
- Chọn file dữ liệu trong thư mục data
- Preview dữ liệu
- Sắp xếp file bằng External Sort

## Cấu trúc project
src/        : source code  
data/       : file dữ liệu đầu vào  
temp/       : file run tạm (không push lên git)

## Cách chạy

Compile:

g++ src/main.cpp -o ext_sort

Run:

./ext_sort
